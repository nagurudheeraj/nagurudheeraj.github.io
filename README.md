https://nagurudheeraj.github.io/

## Prerequisites

Before you begin, ensure you have met the following requirements:

* [Git](https://git-scm.com/downloads "Download Git") must be installed on your operating system.

## Installation

To install, follow these steps:

Linux and macOS:

```bash
sudo git clone https://github.com/nagurudheeraj/nagurudheeraj.github.io.git
```

Windows:

```bash
git clone https://github.com/nagurudheeraj/nagurudheeraj.github.io.git
```
